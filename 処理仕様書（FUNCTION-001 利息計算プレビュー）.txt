000000*****************************************************************
000000* 業務仕様説明（FUNCTION-001 利息計算プレビュー処理）
000000*---------------------------------------------------------------*
000000*    UPDATE         :                                           
000000*        2026/01/19 : 初期化     
000000*---------------------------------------------------------------*
000000* プログラムID : PGM001
000000* 機能ID       : FUNCTION-001
000000*---------------------------------------------------------------*
000000* プログラム概要
000000*---------------------------------------------------------------*
000000* 本機能は、銀行ABCの預金管理システムにおいて、
000000* 有効な預金契約（DB_ACCOUNT_SAVINGS.STATUS = '1'）を対象に、
000000* 現在日時点での利息および合計金額を
000000* プレビュー目的で算出する処理である。
000000*
000000* 本処理は表示専用であり、
000000* 以下の処理は一切行わない。
000000*   ・満期判定
000000*   ・END_DATE の参照
000000*   ・DB 更新
000000*   ・COMMIT / ROLLBACK
000000*
000000*---------------------------------------------------------------*
000000* 対象テーブル
000000*---------------------------------------------------------------*
000000* DB_ACCOUNT_SAVINGS
000000*   - ORDER_ID
000000*   - ACC_ID
000000*   - SAVING_TYPE
000000*   - START_DATE
000000*   - MONEY_ROOT
000000*   - STATUS
000000*
000000* DB_INTEREST_INFO
000000*   - SAVING_TYPE
000000*   - INTEREST_RATE
000000*---------------------------------------------------------------*
000000* 処理フロー概要
000000*---------------------------------------------------------------*
000000* 1. JCL パラメータチェック
000000*   - PARAM = '1' または '3' の場合のみ本機能を実行
000000*
000000* 2. 対象データ取得
000000*   - STATUS = '1' の契約をカーソルで取得
000000*
000000* 3. 経過日数算出
000000*   - CURRENT-DATE を取得
000000*   - START_DATE との差分日数を算出
000000*   - 経過日数 < 0 の場合は 0 に補正
000000*
000000* 4. 利率取得
000000*   - DB_INTEREST_INFO より
000000*     SAVING_TYPE に対応する利率を取得
000000*
000000* 5. 利息計算（単利）
000000*---------------------------------------------------------------*
000000*   利息 = MONEY_ROOT × 利率 × 経過日数 ÷ 365
000000*---------------------------------------------------------------*
000000*
000000* 6. 合計金額算出
000000*   - TOTAL = MONEY_ROOT + INTEREST
000000*
000000* 7. 結果出力
000000*   - DISPLAY により計算結果を出力
000000*
000000*---------------------------------------------------------------*
000000* 注意事項
000000*---------------------------------------------------------------*
000000* - 定期／非定期の区別による処理分岐は行わない
000000* - 利息計算結果は表示目的のみで保持し、
000000*   DB_ACCOUNT_SAVINGS への更新は行わない
000000*****************************************************************
